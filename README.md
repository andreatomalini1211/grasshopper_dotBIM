# grasshopper_dotBIM

This is an initial test developing Grasshopper components to convert **IFC files into the dotBIM format**, inspired by the original [dotBIM project](https://github.com/paireks/dotbim).
Future steps will involve creating components for the reverse process (dotBIM back to IFC).
The `/test` folder contains the original IFC file and the generated results.
